# Условные операторы

Экзаменационные задачи по условным операторам из курса "Поколение Python".

## ��� Решённые задачи

### 1. Шахматная доска
`exam_chess_board.py`  
Определение цвета клетки шахматной доски по её координатам.

### 2. Girls only (футбольная команда)  
`exam_girls_only_football.py`  
Проверка условий для отбора в футбольную команду (пол и возраст).

### 3. Преобразование числа в римские цифры
`exam_number_to_roman.py`  
Конвертация арабских чисел в римские с использованием условий.

### 4. Шахматы: ход слона
`exam_chess_bishop_move.py`  
Проверка возможности хода слона на шахматной доске.

### 5. Шахматы: ход коня
`exam_chess_knight_move.py`  
Проверка возможности хода коня (буквой "Г").

### 6. Шахматы: ход ферзя
`exam_chess_queen_move.py`  
Проверка возможности хода ферзя (по горизонтали, вертикали, диагонали).

## ��� Основные концепции
- Сложные логические условия с `and`, `or`, `not`
- Вложенные условия (if внутри if)
- Работа с координатами и шахматной нотацией
- Преобразование и проверка числовых диапазонов

## ��� Проверяемые навыки
- Формулирование сложных условий на Python
- Работа с шахматной логикой (диагонали, расстояния)
- Обработка пользовательского ввода с валидацией
- Использование математики в условиях (разность координат, модули)

## ♟️ Особенности шахматных задач
1. **Шахматная нотация** — преобразование букв в числа
2. **Цвет клеток** — чётность суммы координат
3. **Диагонали** — равенство разностей координат
4. **Ход коня** — проверка "буквы Г": (|Δx|=1 и |Δy|=2) или (|Δx|=2 и |Δy|=1)

# Шпаргалка: Локальные и глобальные переменные (13.3)

## Основное
- **Локальная переменная** - внутри функции. Видна только там.
- **Глобальная переменная** - вне функций. Видна везде.
- `global` - чтобы изменить глобальную переменную внутри функции.

## Примеры

```python
# Локальная
def func():
    x = 10  # Локальная
    print(x)  # 10

func()
# print(x)  # Ошибка!

# Глобальная
y = 20  # Глобальная
def show():
    print(y)  # 20 (читаем)
show()

# Изменение через global
z = 0
def change():
    global z
    z = 100  # Меняем глобальную
change()
print(z)  # 100
```

## Правило LEGB
Python ищет переменные в порядке:
1. **Local** (локальная функция)
2. **Enclosing** (вложенные функции)
3. **Global** (глобальная)
4. **Built-in** (встроенные)

## Главный вывод
Используй `return`, а не `global`:

```python
# Лучше так:
def add(a, b):
    return a + b
result = add(5, 3)
